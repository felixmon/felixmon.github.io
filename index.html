Hugo 部署完整指南 Preface 网上有很多关于在 github pages 上部署 hugo 的文章，但是不完整，经查阅 hugo 官方文档，我尽量做到完整。 目录 第一步 简要的命令 以下是完整到顺序命令，解释在后文 brew install hugo hugo new site <sitename> cd <sitename> 在 github 上建立一个 <username>.github.io 的repository git clone https://<username>.github.com/<username>.github.io git submodule add https://github.com/EmielH/tale-hugo.git themes/tale hugo new post/hello-world.md hugo server -D 或者 hugo server 打开本地浏览器查阅博客效果 rm -rf public hugo cd public && cp -r * ../<usernname>.github.io cd ../<username>.github.io git add --all git commit -m "it has to be something here" git push origin master 未完待续……
