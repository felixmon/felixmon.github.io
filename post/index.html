Hugo 部署完整指南

Preface

网上有很多关于在 github pages 上部署 hugo 的文章，但是不完整，经查阅 hugo 官方文档，我尽量做到完整。

目录

第一步 简要的命令

以下是完整到顺序命令，解释在后文

  1. brew install hugo
  2. hugo new site <sitename>
  3. cd <sitename>
  4. 在 github 上建立一个 <username>.github.io 的repository
  5. git clone https://<username>.github.com/<username>.github.io
  6. git submodule add https://github.com/EmielH/tale-hugo.git themes/tale
  7. hugo new post/hello-world.md
  8. hugo server -D 或者 hugo server 打开本地浏览器查阅博客效果
  9. rm -rf public
  10. hugo
  11. cd public && cp -r * ../<usernname>.github.io
  12. cd ../<username>.github.io
  13. git add --all
  14. git commit -m "it has to be something here"
  15. git push origin master
